My first post (pinned) 2/11/24

This area is for me to post updates and blogs. This website was built by me using a combination of HTML, CSS, Python, Flask, and MySQL. I intentionally did not use any CSS frameworks, such as bootstrap or tailwind, however I do know how to use them. It was created just to show some of my capabilities. I also wanted to build this as a place to test functionalities and different ways to write code. For this purpose I intend to keep it relatively barebones, just for ease of editing the code and adding in functionalities sort of like a sandbox website for myself. Feel free to test out the current capabilities, sign up, and post your own "blogs" on here and feel free to leave feedback such as comments, and criticisms so I know what I should do better on here, or what was done well. I also would like to hear what ideas on what you would like to see implemented on the site.
